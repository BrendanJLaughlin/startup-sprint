# Sinatra No Database App Template

## Structure

```bash
sinatra-no-db-lab-template/
├── Gemfile
├── Gemfile.lock
├── README.md
├── Rakefile
├── app.rb
├── config
│   └── environment.rb
├── config.ru
├── models
│   └── sample_model.rb
├── public
│   ├── images
│   ├── javascripts
│   └── stylesheets
├── spec
│   ├── sample_model_spec.rb
│   └── spec_helper.rb
└── views
```

## Usage

This template will power labs using Sinatra/Rack. This is for simple apps that do not use ActiveRecord or have a database.
